# r4clinstats: Learn R for Clinical Statistics, Interactively

A swirl-style, learn-by-doing course that teaches clinicians and
clinical researchers to wrangle, tidy, visualise and analyse clinical
data in R. Lessons are interactive 'learnr' tutorials graded by
'gradethis', built on the tidyverse and real-feeling synthetic clinical
datasets.
